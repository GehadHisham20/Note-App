# Note-App
web application allows user to:
1. create account
2. sign in
3. create notes
4. update notes
5. delete notes

## what i use : 
### for frontend
- React.js 
- typed.js
- Joi for validation
- axios for APIs call
- bootstrap
### for backend
- Node.js
- express.js
- mongoose
- Joi for validation
- jsonwebtoken for tokens
- bcrypt for hash password
